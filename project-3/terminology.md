## Create React App
A package that is a template for a basic React app that includes a development environment and tools such as:
- testing
- bundling (with Webpack)
- transpiling (Babel)

Commands for creating a project using Create React App:
```
npx create-react-app app-name
cd app-name
npm start
```
## Component
Function that returns JSX. Conventionally named with PascalCase and called using XML syntax.
